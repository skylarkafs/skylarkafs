> 🤖 WhatsApp bot personal berbasis Go — production-grade, fitur aneh, dibuat pakai cinta.

- **Stack:** `Go` · `whatsmeow` · `SQLite` · `FFmpeg` · `yt-dlp`
- **Pattern:** Modular command handler + media download pipeline
- **Filosofi:** *Ponytail* — reuse sebelum nulis kode baru, stdlib-first

```
SkylarKaf/elaina-bot
├── cmd/        → entry point & graceful shutdown
├── events/     → message dispatcher & dedup
├── handler/    → command modules (SRP per file)
├── media/      → yt-dlp + FFmpeg pipeline
└── store/      → session, DB, keys management
```
