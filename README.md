## 🧙‍♀️ Current Project — Bot Elaina

> *Inspired by Elaina from **Majo no Tabitabi (Wandering Witch)** — a silver-haired witch who travels the world, learning and growing. Bot Elaina is my little companion that does the same.*

<div align="center">

```
  ✦ ·  ˚ · .    ˚ ·   . ✦      ˚   ·  . ✦
       _/_  ☽  Elaina — si penyihir yang berkelana
      /  /\      di antara packet dan socket...
     /--/  \   ✦
    /  /----\     WhatsApp Bot · Go · whatsmeow
   '~~'      ✦ .  ˚  ·   . ✦ ·  ˚
```

</div>

- **Stack:** `Go` · `whatsmeow` · `SQLite` · `FFmpeg` · `yt-dlp`
- **Pattern:** Modular command handler + async media pipeline
- **Filosofi:** *Ponytail* — reuse before rewrite, stdlib-first, no dep baru tanpa alasan jelas

```
SkylarKaf/elaina-bot
├── cmd/        → entry point & graceful shutdown
├── events/     → message dispatcher & deduplication
├── handler/    → command modules (1 file = 1 concern)
├── media/      → yt-dlp + FFmpeg download pipeline
└── store/      → session, database, constants (keys.go)
```
